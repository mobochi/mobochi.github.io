OLED Dark Mode is a tweak that unlocks iOS 13's "OLED" dark mode capability

  • OLED Dark Mode makes the dark, darker. That's it.
  • There are no settings to configure.
  • This tweak hooks directly into iOS's system colors.
  • It is meant to be a lightweight tweak.

NOTE: THIS TWEAK DOES NOT ADD DARK MODE SUPPORT TO APPS THAT DON'T SUPPORT DARK MODE.
