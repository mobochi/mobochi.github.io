MilkyWay 2 is multitasking tweak for iOS13.

    MilkyWay 2 tweak is a free iOS 13 tweak created by @akusio_RR that adds an option to run multiple applications at onces. This verion only support iOS 13.x and doesn't require any configuration. Activate multitasking and run multiple apps in small windows.
